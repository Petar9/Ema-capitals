# Куиз за Столици (Capital Cities Quiz)

Интерактивна игра за познаване на столици.
Проект на Ема Живкова, 3Д клас, 36СУ.

## Как се играе
1. Стартирайте `index.html`.
2. Изберете правилната столица за държавата.
3. Разгледайте снимките на забележителностите.
4. Съберете максимален брой точки!

## Технологии
- HTML5
- Tailwind CSS (CDN)
- JavaScript
